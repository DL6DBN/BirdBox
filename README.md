# BirdBox
My version of a RaspberryPi Bird Box

On the Raspberry Pi Homepage I found a tutorial about a [bird box](https://www.raspberrypi.org/learning/infrared-bird-box/) watching camera. As I had an PiNoIR camera and a Raspberry Pi 2 at hand for no other purposes, I started this project in mid February. I hoped this won't be to late for some birds to find a home in the box.
